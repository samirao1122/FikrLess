# Mood Images

This folder should contain 6 mood images for the mood selection feature.

## Required Images:

1. `happy.png` - For "Happy" mood
2. `very_happy.png` - For "Very Happy" mood
3. `calm.png` - For "Calm" mood
4. `sad.png` - For "Sad" mood
5. `anxious.png` - For "Anxious" mood
6. `tired.png` - For "Tired" mood

## Image Specifications:

- Recommended size: 64x64 pixels or higher (will be scaled down)
- Format: PNG with transparency preferred
- The images will be displayed in the mood card on the home screen and in the mood selection screen

## Adding Images:

1. Place the mood images in this folder with the exact names listed above
2. Run `flutter pub get` to ensure assets are registered
3. The app will automatically use these images when displaying moods
